# Product Page

## Start Project
```
git clone https://github.com/viniarifianir16/assignment-fe-viniarifianirohmat.git
npm install
npm run dev
```
    
## Features
- Search Product
- Filter Category
- Sort Price 
- Responsive Mobile

## Library
- TailwindCSS
- Flowbite React UI
- React Icons
